# angular-tests
Unit Tests for Angular using Jasmine and Karma

###Steps to run the project:
    1) Install node
    2) Run npm install grunt-cli  -g
    3) Run npm install karma-cli -g
    4) Go to project folder from command line and run npm install
    5) Run node server.js
 
###Steps to run the tests:
    1) Using karma: karma start karma.conf.js
    2) Using grunt: grunt
    3) Standalone Jasmine Runner: Open the specRunner.html file in browser.
